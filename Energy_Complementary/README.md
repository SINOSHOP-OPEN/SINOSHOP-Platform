# ⚡ Energy_Complementary — 多能互补调度系统

> **实现海洋城市平台的能源自给。**
> 集成漂浮式光伏、垂直轴风电、潮流能、波浪能，构建分布式海上超大型发电站。

## 核心内容

### /Forecasting_Data — 功率预测数据集
- 光伏、风电、潮流能、波浪能的时序预测数据
- 海洋盐雾环境衰减系数修正

### /Dispatch_Algorithm — 智能调度算法
- 日前/实时多时间尺度优化调度
- AI 驱动的功率预测与负荷匹配

### /Hydrogen_Storage — 氢储能循环
- “电-氢-电”效率模型
- 海上绿氢规模化制备与储运技术
