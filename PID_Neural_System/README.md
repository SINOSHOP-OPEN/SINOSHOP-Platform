# 🧠 PID_Neural_System — 全悬浮PID神经控制系统

> **海洋城市平台的“神经系统”：实现全工况下毫米级姿态协同调节。**
> 这是 SINOSHOP-OS 的核心控制中枢，将分布在全平台数以千计的传感器数据实时融合，计算最优姿态调整指令，驱动推进器与压载水舱联动。

## 系统架构

中央控制单元 (CCU) 包含姿态解算引擎、波浪预测模型、推力分配算法，底层由多传感器融合滤波器 (EKF/UKF) 驱动。传感器网络覆盖 IMU 阵列、X-band 波浪雷达阵列和 ADCP 潮流监测网络。

## 核心模块

### /preprocessing — 传感器数据预处理
- IMU 去噪：基于 Allan 方差的噪声参数辨识
- 波浪雷达点云解析：海面波高场反演算法
- ADCP 流速剖面预处理：异常值剔除与空间插值

### /simulation — MATLAB/Simulink 仿真模型
- 全平台六自由度动力学模型 (Cummins 方程)
- PID + 前馈复合控制器 (Simulink)
- 极端海况测试场景库 (18米波高、台风眼墙过境)
- 硬件在环 (HIL) 接口定义

### /control_law — 控制律伪代码与文档
- 推进器推力分配优化算法
- 压载水舱调载策略
- 多模块协同控制通信协议 (CAN/EtherCAT)

## R16 协议关联
所有控制指令与传感器数据流必须遵循 SINOSHOP R16-SDK 编码标准。未经 R16 认证的控制器接入将触发自动隔离保护。
