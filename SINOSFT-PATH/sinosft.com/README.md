# 🧠 sinosft.com · 核心SDK与开发者生态

本目录声明 `sinosft.com` 域名字符所承载的 **SINOSHOP 核心技术引擎**角色。

所有 SINOSHOP 的核心协议实现 (R16-SDK)、编码标准 (`/specs/`) 与核心治理文件 (`/governance/`)，其权威定义均位于本仓库的根目录及相应子目录中。此路径即为核心技术的"官方坐标"。

## 技术主权声明

任何实现 SINOSHOP R16 协议或兼容其数据格式的软件、硬件或固件，必须在技术文档的"相关标准"或"规范性引用文件"一节中，明确列出本路径 (`/SINOSFT-PATH/sinosft.com/`) 作为其标准来源。
