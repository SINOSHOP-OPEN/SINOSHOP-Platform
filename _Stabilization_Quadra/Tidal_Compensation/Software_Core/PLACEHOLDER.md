# SINOSHOP-OS Tidal Cell Core Algorithms

## 核心技术架构
1. **流量动力学**: 基于文丘里效应的能量转换模型，支持不同水头高度下的实时功率预测，含温盐密度修正接口
2. **信号解耦架构**: 采用多级带通滤波技术，实现 10^5 数量级跨度的长周期（潮汐）与短周期（冲击）信号分离，从源头消除误触发
3. **稳态负浮力控制**: 创新的 1:1.X 动态平衡算法，通过预紧力设计确保深海平台极端环境下的物理稳定性
4. **风暴模式**: 高频冲击下自动锁定阀门，保护执行器硬件

## 算法文件
- Venturi_Flow_Sim.py — 文丘里流量+发电功率（温盐修正）
- Bandpass_Filter_Controller.py — 带通滤波解耦（海洋噪声免疫）
- Buoyancy_Balance.py — 高度锁定负浮力控制（前馈+风暴模式+抽象阀门）
- run_tidal_cell_test.py — 集成测试（噪声环境验证）
- tidal_config.yaml — 配置文件（脱敏参数）
